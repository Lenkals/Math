Why Maxima & Minima

The Second Derivative Test

A critical point is where f′(x) = 0 — the tangent line is horizontal (slope = 0). The curve is momentarily flat there.

▲ Local Maximum: f′ changes + → 0 → −, and f″(x) < 0 (concave down ∩ — hilltop shape).

▼ Local Minimum: f′ changes − → 0 → +, and f″(x) > 0 (concave up ∪ — valley shape).

⚡ They appear at the ACTUAL peak/valley of f(x), not near x=0. The dots in the graph are placed at the exact (x, f(x)) of the extremum.

📐 f′(x) and f″(x) are drawn scaled to fit same view — check hover panel for true values.
